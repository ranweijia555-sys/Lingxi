# 塔罗体系导论

## 概述

塔罗牌由 78 张牌组成，分为 22 张大阿尔卡纳（Major Arcana）和 56 张小阿尔卡纳（Minor Arcana），每一张牌都蕴含独特的数字能量（灵数）与四元素属性（火、水、风、土）。

大阿卡纳牌偏向于精神与灵魂层次的解读，影响的范围较大；而小阿卡纳则是偏向实质的状况，影响的范围较少。这是因为大阿卡纳的人物牌都是以历史人物或者神话人物为原型，有人格、心理、社会定位多方面的资料可以参考。

## 二、二分法 / 三分法

### 二分法
- **俗（1-10）**：物质躯壳层面的完善
- **圣（11-20）**：精神层面的成长
- **特殊：0 愚者** — 不在划分范围内

### 三分法（出自《78℃智慧》）
- **身（1-7 魔术师→战车）**：肉体与外在世界
- **心（8-14 力量→节制）**：情感与意志
- **灵（15-21 恶魔→世界）**：精神与超越
- **特殊：0 愚者** — 不在划分范围内

## 三、小阿尔卡纳：四元素 + 灵数 + 宫廷牌

### 核心解牌公式

**元素 + 灵数 = 核心能量**

- **元素**：决定问题聚焦的领域和行动方式
- **灵数**：决定能量的阶段和发展状态

### 1. 四元素对应花色

| 花色 | 元素 | 关注核心 | 代表领域 |
|---|---|---|---|
| 权杖 Wands | 火 | 想不想做 | 激情、意志力、行动力 |
| 圣杯 Cups | 水 | 感觉好不好 | 情感、人际关系、直觉 |
| 宝剑 Swords | 风 | 想得对不对 | 理智、思想、冲突 |
| 星币 Pentacles | 土 | 能不能落地 | 物质、金钱、身体健康 |

### 2. 灵数 1-10 能量

| 数字 | 含义 |
|---|---|
| 1 (Ace) | 潜能、新开始、种子 |
| 2 | 平衡、合作、选择 |
| 3 | 创造、团队、初步成果 |
| 4 | 稳定、停滞、基础 |
| 5 | 挑战、冲突、改变 |
| 6 | 调和、分享、过渡 |
| 7 | 内省、评价、评估 |
| 8 | 管理、专注、移动 |
| 9 | 成熟、接近完美、总结 |
| 10 | 完成、圆满、新循环开始 |

### 3. 宫廷牌（Court Cards）

| 牌 | 元素 | 阶段 | 关键词 |
|---|---|---|---|
| 侍从 Page | 风 | 探索期 | 好奇、学习、收集信息 |
| 骑士 Knight | 火 | 推进期 | 行动、执行、快速推进 |
| 王后 Queen | 水 | 守护期 | 滋养、情感、守护资源 |
| 国王 King | 土 | 建构期 | 稳定、权威、构建现实 |

> 注：另有传统体系把 Page=土 / Knight=火 / Queen=水 / King=风。本项目采用上述阶段论体系。

### 4. 宫廷牌解读公式

**人物元素 + 牌组元素 = 综合含义**

举例（宝剑组 = 风元素）：
- 宝剑侍从（风+风）= 思维信息的探索者
- 宝剑骑士（风+火）= 思维目标的推进者
- 宝剑王后（风+水）= 思维资源的守护者
- 宝剑国王（风+土）= 思维成果的建构者

## 四、灵数归元体系

### 核心概念
塔罗牌中的数字都遵循 1-9 的循环，大于 9 的数字需要"归元"到 1-9。

### 归元方式
- 个位 + 十位相加
- 若结果仍大于 9，继续相加直到 1-9

### 举例
- 15 号牌 "恶魔"：1 + 5 = **6**
- 19 号牌 "太阳"：1 + 9 = 10 → 1 + 0 = **1**
- 22 号牌 "愚者"（在某些体系中）：2 + 2 = **4**

### 灵数 1-9 深层含义

| 灵数 | 深层含义 |
|---|---|
| 1 | 源头、启动、新阶段 |
| 2 | 二元、流动、寻求平衡 |
| 3 | 结合、产出、初步成果 |
| 4 | 结构、固化、建立秩序 |
| 5 | 动荡、冲突、打破平衡 |
| 6 | 秩序、回归、承担责任 |
| 7 | 突围、博弈、主动出击 |
| 8 | 沉淀、成长、由量变到质变 |
| 9 | 巅峰、孤独、完成使命 |

## 五、占卜时的元素快速判断

在占卜中，可以通过牌组的元素分布快速判断核心问题：

- **权杖多** → 事业、行动、热情主导
- **圣杯多** → 感情、情绪、心理主导
- **宝剑多** → 理智、冲突、压力主导
- **星币多** → 物质、财运、身体主导

### 数与元素结合示例

- **权杖三**（火+3）：充满活力的团队合作，向外拓展
- **圣杯五**（水+5）：情绪上的丧失或挑战
- **宝剑十**（风+10）：理智上的彻底终结
- **星币八**（土+8）：勤奋的积累财富和技能
