# 灵犀 Lingxi

A multi-source RAG system for symbolic divination integrating Tarot, Western Astrology, and Chinese BaZi.

## 项目简介
灵犀是一个整合塔罗、西方占星、东方八字的多源符号占卜 RAG 助手。

## 技术栈
- Frontend: Next.js + TypeScript + Tailwind
- Backend: FastAPI + Python
- LLM: GPT-4o + DeepSeek V3
- Vector DB: ChromaDB
